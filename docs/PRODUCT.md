# YSOAM — Product Vision

**Platform:** [ysoam.com](https://ysoam.com)  
**Market:** India — fleet operators, mid-market to enterprise  
**Currency:** INR (₹) · GST-aware billing

---

## Vision

YSOAM is the profitability-first fleet platform for Indian transport operators. One system replaces GPS apps, Excel billing, and fuel spreadsheets — with trip, vehicle, and route margin analysis built in.

**Hero metric:** See which trips make money in **5 clicks**.

---

## Problem

Without YSOAM, fleet operators:
- Don't know which routes are profitable
- Re-enter data across GPS, billing, and fuel systems
- Invoice customers manually (hours per week)
- Cannot tie driver performance to trip profit
- Cannot answer "which vehicle should I assign?" with data

---

## Solution

| Capability | Value |
|------------|-------|
| Unified platform | GPS + ERP + billing in one login |
| Profitability engine | Trip, vehicle, route margin analysis |
| Auto-billing | Invoices generated from completed trips |
